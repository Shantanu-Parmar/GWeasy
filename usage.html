Usage

OSDF Server-based (Recommended)

  1. Refresh the list if needed; it will fetch the data index in real-time.
  2. Select which detector (Hanford, Livingston, Virgo, or KAGRA) you want to work with.
  3. Choose the frame type/channel to fetch from.
  4. Available time segments will load in the last column; select one or multiple.
  5. Select an output directory.
  6. Press "Download Data".
  7. Toggle "Start/Stop Execution" to pause/play as needed.

NDS (Legacy Server) based

  1. Select the detector.
  2. Choose which channel group to use (e.g., Physical Environment Monitoring is PEM).
  3. Select the specific channel.
  4. Time segments will load; select one or multiple.
  5. Select an output directory.
  6. Press "Download Data".
  7. Toggle "Start/Stop Execution" as needed.

Batch Mode (for Listed Time Segments and Channels)

  1. Select CSVs for time segments and channels.
  2. Choose the channel to fetch first.
  3. Choose time segments to work with; press "Deselect Processed" to toggle already fetched .gwf time segments.
  4. Select an output directory.
  5. Press "Start Execution".
