Usage

Running a Test in the Gravfetch Tab

  1. Select test.csv as the time segment input.
  2. Select 4KCHANS.csv as the channels input.
  3. Optionally, specify an output directory (default: gwfout).
  4. Click "Start Execution" to begin. A completion message will appear in the terminal (~5-7 minutes depending on segments and channels).

For faster testing, start with a single time segment and channel.

Running Omicron Analysis

  1. Switch to the Omicron tab after generating .gwf files in Gravfetch.
  2. Select a channel and sampling rate first (due to a UI issue, configure these before other fields).
  3. Choose the .ffl file from the gwfout directory or your output path.
  4. Use the "Custom Segments" feature to manually specify time segments if needed.
  5. Specify an output directory (default: Omicronout).
  6. Click "Save Config" to generate the configuration file.
  7. Click "Start Omicron" to begin processing. A completion message will appear upon success.
