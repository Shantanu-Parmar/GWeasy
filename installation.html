Installation Guide

GWeasy offers multiple installation methods for Windows and Linux. Choose the option that best fits your workflow. All methods are designed for simplicity and minimal setup.

Windows Installation

GWeasy supports Windows through WSL integration, standalone EXE, and zip-based runs. Follow the steps below for your preferred method.

Windows + WSL (Recommended)

  1. Fetch WSL Copy: Download the pre-configured GWeasy WSL image from Zenodo (unless using your own WSL; see Custom WSL section below).
  2. Setup WSL on Your PC: Enable WSL via Windows Features (search "Turn Windows features on or off" > check "Windows Subsystem for Linux"). Install Ubuntu from Microsoft Store.
  3. Import WSL Image: Open PowerShell as admin and run: wsl --import GWeasyWSL C:\WSL\GWeasy path\to\GWeasywsl.tar.
  4. Launch and Run: Start WSL with wsl -d GWeasyWSL, then execute ./GWeasy.
Custom WSL:

If using your own WSL distro, install dependencies (Miniconda, GW libraries) manually and clone the repo: git clone https://github.com/shantanu-parmar/GWeasy.git.

Standalone EXE

  1. Fetch Conda Copy: Download the Conda environment from Zenodo.
  2. Setup Conda on Your PC: Install Miniconda from official site.
  3. Fetch Installer and Run: Download the installer script from GitHub Releases and run it to set up the environment.
  4. Standalone EXE Ready: Fetch the built EXE from Releases, double-click to run.

Zip-Based Run with Built EXE

  1. Download Zip: Get the pre-built zip from GitHub Releases.
  2. Extract and Run: Unzip the file and double-click the EXE to launch GWeasy.
  3. Python Script Option: For custom runs, use python main.py after installing dependencies.
